# 🎙️ Python Voice Assistant

A smart voice-controlled assistant built using Python that can perform various tasks through voice commands. The assistant listens to user input, processes the command, and responds using speech synthesis. It can search the web, provide information from Wikipedia, open applications, play music, send emails, and perform several system operations.

---

## 📌 Features

- 🎤 Voice Recognition
- 🔊 Text-to-Speech Responses
- 📚 Wikipedia Search
- 🌐 Open Websites (Google, YouTube, etc.)
- 🔍 Google Search via Voice Commands
- 🎥 YouTube Search via Voice Commands
- 🎵 Music Playback
- ⏰ Current Time Announcement
- 📧 Email Sending Functionality
- 📝 Open Notepad
- 🧮 Open Calculator
- 💻 Open Command Prompt
- 🔄 System Restart
- ⚡ System Shutdown

---

## 🛠️ Technologies Used

- Python 3.x
- SpeechRecognition
- pyttsx3
- PyAudio
- PyAutoGUI
- Wikipedia API
- SMTP Protocol
- WebBrowser Module
