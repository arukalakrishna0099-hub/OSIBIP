# 🌦️ Simple Weather App

A simple command-line weather application built with Python that fetches real-time weather information using the OpenWeatherMap API.

This project was developed as part of my Python Programming Internship at Oasis Infobyte.

---

## Features

- 🌍 Search weather by city name
- 🌡 Current temperature
- 🤗 Feels like temperature
- 💧 Humidity
- 🌀 Atmospheric pressure
- 💨 Wind speed
- ☁ Weather conditions
- ⚠ Error handling for invalid city names
- 🔄 Multiple city searches in one session
- ⏱ Network timeout handling

---

## Technologies Used

- Python 3
- Requests Library
- OpenWeatherMap API

---

## Installation

Clone the repository

git clone https://github.com/yourusername/simple-weather-app.git


Move into the project directory

cd simple-weather-app


Install dependencies

pip install -r requirements.txt


## Get Your API Key

1. Create an account on OpenWeatherMap.
2. Generate your free API key.
3. Replace:

API_KEY = "YOUR_API_KEY"

with your own API key.
