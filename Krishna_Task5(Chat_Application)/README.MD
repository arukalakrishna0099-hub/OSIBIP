# 💬 Python Chat Application

A simple command-line chat application built using Python that enables two users to exchange messages in real time using a client-server architecture. This project demonstrates the fundamentals of socket programming, networking, and concurrent communication.

## 📌 Project Overview

This project implements a basic chat system where:

- One user acts as the **Server**
- Another user acts as the **Client**
- Messages are exchanged instantly through TCP sockets
- Communication continues until either user exits the chat

It is an excellent beginner project for learning Python networking concepts and client-server communication.
