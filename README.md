## 2. BMI Calculator

### Description
A command-line application that calculates Body Mass Index (BMI) using user-provided height and weight values.

### Features
- User-friendly input
- BMI calculation
- Health category classification
- Input validation

### BMI Categories

| BMI Range | Category |
|------------|-----------|
| Below 18.5 | Underweight |
| 18.5 - 24.9 | Normal Weight |
| 25 - 29.9 | Overweight |
| 30 and Above | Obese |

### Technologies Used
- Python

### Learning Outcomes
- Mathematical computations
- Conditional statements
- User input handling
